ListExtender.js

Documentation

Getting Started

The first step is to import ListExtender.js and any javascript using ListExtender into your html file. Make sure you import it before any javascript files that depend on it.

<script defer type='text/javascript' src='js/ListExtender.js'></script>
<script defer type='text/javascript' src='js/myScript.js'></script>

The next step is to create your ListExtender object.

const myList = new ListExtender()

Next, add a placeholder for your input elements and a new input element to your list object.

myList.setPlaceholder('Placeholder Text')

Finally, append your list to some parent container in the DOM

myList.appendTo('#containerId')

API

Constructor:

new ListExtender(options)

The ListExtender constructor takes an optional parameter, which is an options object. The properties of the options parameter determine the behaviour of the list.

The options object uses the following properties:

Property Default Value Description
isUnordered true Determines whether the list is unordered or not
allowReorder true Determines whether users can drag and drop list items
showDeleteButton false Determines whether the delete button shows on the list

All properties are optional and if not included will be replaced with the default value.

Variables:

Name Description
maxSize The maximum number of entries in the list
simpleLight A light theme for list objects
simpleDark A dark theme for list objects
cream A light brown theme for list objects
hacker A dark/lime theme for list objects
underwater A blue theme for list objects
MLA A simple double spaced theme for list objects
PuTTY A shell theme for list objects
emoji A simple theme with emoji pointers for list objects
